# Password_generator_0-1
A dynamic password generator built with HTML, CSS, and JavaScript that creates secure random passwords based on user preferences. Users can adjust the length using a slider and choose character types such as uppercase, lowercase, numbers, and symbols through checkboxes. The password is instantly generated and displayed when the button is clicked.❤️
